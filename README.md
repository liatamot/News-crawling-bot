# News-crawling-bot

## 1. Project Info

### Overview

- 파이썬을 활용하여 크롤링, API 연동 등의 기능을 구현한 정보 자동화 프로젝트


## 2. Data descrption

### Dataset overview

- `Selenium Library`와 사용자 input data를 통해 검색된 `구글 뉴스 기사 데이터`(제목, 말머리, 링크 등)

- 수집된 정보를 `Slack API`를 통해 연결함으로써 원하는 정보를 간편하게 관리하고 아카이빙
